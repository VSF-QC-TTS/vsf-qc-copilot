# Story 04.01 - Connector List

> Epic: 4 - Target API Connectors
> Depends on: 03.03
> Backend contract source: `server/API_TREE.md`

## Story Header

**As a** QC member,
**I want** to list connectors for a project,
**so that** I can see which target APIs are available for evaluation.

## In Scope

- [WEB] Project-scoped connector list.
- [WEB] Method, URL preview, active status, and created date.
- [WEB] Row navigation to detail/edit.

## Out of Scope

- Secret value display.

## Deferred

- Connector grouping.

## API/Data Contract

```text
GET /api/v1/projects/{projectPublicId}/target-api-connectors?page=0&size=20
Response: PageResponse<TargetApiConnectorListItemResponse>
```

## UI States

- Empty connector state with create CTA.
- Long URL truncation.
