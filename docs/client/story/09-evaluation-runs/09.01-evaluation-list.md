# Story 09.01 - Evaluation List

> Epic: 9 - Evaluation Runs
> Depends on: 04.03, 06.03, 08.05
> Backend contract source: `server/API_TREE.md`

## Story Header

**As a** QC member,
**I want** to list evaluation runs in a project,
**so that** I can inspect previous and active quality checks.

## In Scope

- [WEB] Project-scoped evaluation run DataTable.
- [WEB] Status filters.
- [WEB] Row navigation to run detail.

## Out of Scope

- Cross-project run list.

## Deferred

- Dashboard aggregate run feed.

## API/Data Contract

```text
GET /api/v1/projects/{projectPublicId}/evaluation-runs?page=0&size=20&sort=createdAt,desc
Response: PageResponse<EvaluationRunListItemResponse>
```

## UI States

- Running rows show animated status badge.
- Empty run state with start evaluation CTA.

## Acceptance Criteria

### AC 1 - Runs are project-scoped
