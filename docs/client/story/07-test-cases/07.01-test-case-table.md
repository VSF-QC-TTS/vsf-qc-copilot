# Story 07.01 - Test Case Table

> Epic: 7 - Test Cases
> Depends on: 06.03
> Backend contract source: `server/API_TREE.md`

## Story Header

**As a** QC member,
**I want** to view test cases inside a dataset,
**so that** I can review coverage before approving or evaluating.

## In Scope

- [WEB] Dataset-scoped test case DataTable.
- [WEB] Status filter.
- [WEB] Toolbar actions for create, import, and AI generate when dataset is DRAFT.

## Out of Scope

- Bulk delete/toggle until backend batch endpoints exist.

## Deferred

- Batch actions.

## API/Data Contract

```text
GET /api/v1/datasets/{datasetPublicId}/test-cases?page=0&size=20&status=ACTIVE
```

## UI States

- Empty test case state.
- Read-only action state for APPROVED/ARCHIVED datasets.
