## Story Header

**As a** QC member,
**I want** to list project datasets,
**so that** I can choose or prepare test data for evaluation.

## In Scope

- [WEB] Project-scoped dataset DataTable.
- [WEB] Status filters for DRAFT, APPROVED, ARCHIVED.
- [WEB] Test case count display.

## Out of Scope

- Inline test case management.

## Deferred

- Dataset duplication.

## API/Data Contract

```text
GET /api/v1/projects/{projectPublicId}/datasets?page=0&size=20&status=DRAFT
Response: PageResponse<DatasetListItemResponse>
```

## UI States

- Empty dataset state.
- Status badge per dataset.
