# Story 03.02 - Create Project

> Epic: 3 - Projects
> Depends on: 03.01
> Backend contract source: `server/API_TREE.md`

## Story Header

**As a** QC member,
**I want** to create a project,
**so that** I can group connectors, requirements, datasets, and evaluations.

## In Scope

- [WEB] Create project dialog.
- [WEB] Name and optional description fields.
- [WEB] Navigate to created project detail.

## Out of Scope

- Project members.

## Deferred

- Project cloning.

## API/Data Contract

```text
POST /api/v1/projects
Request: { name, description? }
Response: ProjectResponse
```

## UI States

- Validation error for short/empty name.
- Loading submit state.
- Success closes dialog and invalidates list.

## Acceptance Criteria

### AC 1 - Create navigates to project

```text
Platform: [WEB]
Actor: QC member

Given  the member submits a valid project name
When   POST /api/v1/projects succeeds
Then   the project list cache is invalidated
And    the member navigates to the new project detail route
```
