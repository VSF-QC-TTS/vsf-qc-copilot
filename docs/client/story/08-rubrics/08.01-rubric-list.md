# Story 08.01 - Rubric List

> Epic: 8 - Rubrics
> Depends on: 03.03
> Backend contract source: `server/SERVER_CONTEXT.md`

## Story Header

**As a** QC member,
**I want** top-level access to my rubrics and templates,
**so that** I can manage reusable evaluation rules independently of a single project.

## In Scope

- [WEB] `/rubrics` page.
- [WEB] My Rubrics tab.
- [WEB] Templates tab with clone action.

## Out of Scope

- Project-only rubric management.

## Deferred

- Sharing rubrics between users.

## API/Data Contract

```text
GET /api/v1/rubrics?page=0&size=20
GET /api/v1/rubrics/templates?page=0&size=20
POST /api/v1/rubrics/{rubricPublicId}/clone
```

## UI States

- Empty my-rubrics state.
- Template clone loading.

## Acceptance Criteria
